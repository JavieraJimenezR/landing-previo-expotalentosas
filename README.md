# Landing Previo Expo Talentosas

Landing page de pre-lanzamiento para la feria laboral Expo Talentosas 2026.

## 🚀 Características

- **Hero Section**: Diseño retro con ventana de navegador y captura de emails
- **Cómo Funciona**: 3 pasos del proceso con IA
- **Tutorial**: Guía visual con video explicativo
- **Diseño Responsive**: Optimizado para todos los dispositivos
- **Animaciones**: Transiciones suaves y animaciones modernas

## 🛠️ Tecnologías

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (iconos)

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Vista previa de la compilación
npm run preview
```

## 🎨 Estilo Visual

El diseño mantiene la identidad de Expo Talentosas con:

- **Colores principales**: 
  - Azul primario: `hsl(220, 90%, 56%)`
  - Accent/Violeta: `hsl(280, 100%, 70%)`
  - Rosa: `hsl(330, 100%, 71%)`
- **Tipografía**: Inter (bold y moderna)
- **Estilo**: Retro con bordes definidos y sombras pronunciadas

## 📱 Secciones

1. **Header**: Navegación fija con logo y CTA
2. **Hero**: Ventana retro con información del evento y stats
3. **Cómo Funciona**: Cards con los 3 pasos principales
4. **Tutorial**: Video y lista de beneficios
5. **CTA Final**: Segundo formulario de captura
6. **Footer**: Enlaces e información de contacto
